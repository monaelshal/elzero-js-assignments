# JavaScript Zero Assignments

## Description
This repository contains all the assignments and exercises from the JavaScript Zero course. 
It covers the basics of JavaScript, including variables, data types, operators, loops, functions, and more.

## Languages and Tools
- JavaScript
- HTML & CSS (if used in some exercises)
- VS Code or any code editor

## Assignments / Topics
- Introduction to JavaScript
- Data Types (String, Number, Boolean, etc.)
- Variables (let, const, var)
- Operators (+, -, *, /, %, etc.)
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Functions (declaration, expression, arrow functions)
- Arrays and basic array methods
- Objects and key-value pairs
- DOM manipulation (if any assignment covers it)

## How to Run
1. Clone the repository: https://github.com/monaelshal/elzero-js-assignments.git
2.  Open the `.html` or `.js` files in your code editor.
3. Run the `.html` files in the browser or execute `.js` files with Node.js (if applicable).

## Notes
- Each folder contains assignments for a specific topic.
- Files are named according to the assignment topic for easy navigation.
